## NODE JS VIDEO API ##

 - Simple REST API that stores information about videos and uses authentication (via middleware) to retrieve data.

- [x] Uses NodeJS, Express & MongoDB for core interactions
- [x] MVC approach
- [x] Has custom middleware for authentication (via JSON Web Tokens) 
- [x] Joi for validation
- [x] Mongoose to create & validate models
- [x] Middleware for authentication
- [X] All CRUD methods built in (GET, POST, PUT, DELETE)
